# Ocean.Breeze.Beach.Club 🌴

--------

# 🏖️ Python System

Um projeto de simulação desenvolvido em Python que recria o funcionamento de um clube de praia.

O sistema representa diferentes áreas de uma empresa, incluindo reservas, hospedagem, restaurantes, serviços, clientes, eventos e controles financeiros.

---------

# 📌 Módulos do Sistema

## 🏖️ Reservas de Cabanas
Sistema para gerenciamento de reservas e disponibilidade de cabanas.

## 🍔 Restaurante
Controle de pedidos, produtos e valores.

## 🍹 Bar da Praia
Simulação de atendimento e pedidos de bebidas.

## 🏨 Hotel
Gerenciamento de hospedagem e cálculo de diárias.

## 🏄 Aluguel de Equipamentos
Controle de aluguel de equipamentos de praia.

## 👥 Clientes
Cadastro e gerenciamento de clientes.

## 📅 Eventos
Organização de eventos do clube.

## 💰 Financeiro
Controle de entradas e saídas.

## ⭐ Fidelidade
Sistema de pontos e benefícios.

------

# 🐍 Tecnologias utilizadas

- Python
- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Listas e dicionários
  
-------

# 🎯 Objetivo

Desenvolver uma coleção de sistemas utilizando Python básico, simulando situações reais de uma empresa e aplicando conceitos de programação.
